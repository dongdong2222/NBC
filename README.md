# 5번 과제
## 개요
언리얼에서 c++로 간단한 로그를 찍는 과제입니다.
구현 사항은 다음과 같습니다.
- 시작점(0,0) 있는 게임 캐릭터가 랜덤하게 10회 이동 합니다.
- 이동 이벤트를 1/2 확률로 실패합니다.
- 이동 시 출력해야할 내용
  - 이동 위치
  - 움직인 거리
- 완료된 후 전체 이벤트 성공횟수와 이동거리를 출력합니다.
